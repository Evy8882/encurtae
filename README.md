## 📖 Sobre o projeto

O **Encurtaê** é uma aplicação web de encurtamento de URLs desenvolvida para transformar links longos em URLs curtas e fáceis de compartilhar.

O projeto foi criado com foco em:

- ⚡ Performance
- 🎯 Simplicidade
- 🧩 Separação entre frontend e backend
- 📚 Aprendizado de APIs e persistência de dados

---

## ✨ Funcionalidades

- ✅ Encurtamento de URLs
- ✅ Redirecionamento automático
- ✅ Geração de códigos únicos
- ✅ Exclusão de links
- ✅ Armazenamento local
- ✅ API REST em Go
- ✅ Interface moderna em React
